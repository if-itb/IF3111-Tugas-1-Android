Nama Aplikasi: Tangkap Jerry!
=======================
1. Jalankan program dengan memilih-nya dari launcher.
2. Aplikasi akan otomatis menampilkan posisi Jerry di saat aplikasi dijalankan.
3. Jika diperlukan, anda dapat mendapatkan posisi Jerry dengan menekan tombol
   "Tanya Spike". Karena aplikasi ini user-friendly, maka sebenarnya tombol ini
   tidak diperlukan.
4. Tangkap Jerry dengan menekan tombol "Laporkan Jerry"
5. Arah panah biru merupakan penunjuk arah mata angin, panah tersebut menunjuk
   ke arah utara.
6. Posisi Jerry ditampilkan sebagai marker berwarna merah dan posisi anda
   ditampilkan sebagai bulatan biru.

Selamat menangkap Jerry!
