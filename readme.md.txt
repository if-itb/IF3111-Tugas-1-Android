Tugas Besar 1 PBD
Pada tugas besar satu ini mahasiswa diminta untukk membuat aplikasi berbasis android
untuk menerima data dari server, memetakannya ke google map, scan barcode, dan mengirimkan data hasil scan
ke server. Server yang digunakan untuk mendapatkan data adalah http://167.205.32.46/pbd/api/track?nim=13512078
sedangkan untuk mengirim data adalah http://167.205.32.46/pbd/api/catch

Cara menjalankan Aplikasi
1. Pastikan layar yang digunakan berkisar 7-8 inch karena layout belum mendukung layar lainnya
2. Pilih button Start Tracking Jerry untuk melacak jerry pada google Map, pada google map ditampilkan
	waktu yang tersisa, posisi jerry, posisi kita, dan compass
3. Untuk menginput QR code harus masuk kembali ke halaman utama
4. Pilih menu Input QR Code
5. arahkan kamera ke QRcode yang ingin di Scan
6. Akan ditampilkan hasil dari server (OK, parameter missing, dsb)
